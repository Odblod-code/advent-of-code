# Advent of Code - Prácticas con TDD

Este repositorio contiene soluciones a ejercicios del evento [Advent of Code](https://adventofcode.com/), con un enfoque centrado en la escritura de código utilizando **Test-Driven Development (TDD)** en **TypeScript**.

El objetivo principal es practicar resolución de problemas, mejorar la calidad del código, y reforzar habilidades en pruebas automatizadas, manteniendo buenas prácticas de diseño.

Cada día o reto resuelto incluye:
- Código de solución modular
- Pruebas automatizadas escritas con Jest
- Enfoque iterativo guiado por los tests

Este repositorio sirve como muestra de mis hábitos de desarrollo y forma parte de mi aprendizaje continuo en programación.

---

## Tecnologías usadas

- TypeScript
- Jest
- Node.js
- (opcionalmente) Angular en algunos retos
